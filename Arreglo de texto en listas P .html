En este capitulo encontraras como dar formato alistas enumeradas, listan con viñetas y otra variedad de listas sangradas. Tambien vera como las etiquetas HTML para crear listas se pueden utilizar cualquier otro tipo de Sangrado que desee en sus paginas Web

Los tres tipos de listas HTML

  
1.- La lista enumerada de la parte superior la cual se llama Lista ordenada. Comienza con la etiqueta OL y termina con la etiqueta de cierre /OL. Los numeros y los saltos aparecen automaticamente con cada etiqueta LI y toda la lista queda sangrada

  
2.- La lista con viñetas se llama Lista no ordenada. Se abre con la etiqueta UL y se cierra con /UL. Se ve muy parecida a una lista ordenada, con la excepcion de que en vez de numeros, con cada etiqueta LI aparecen iñetas

  
3.- La lista de terminos y sus significados se llama una lista de definicion. Comienza con la etiqueta LD y termina con /DL. La etiqueta DL va a delante de cada termino por definir con la etiqueta DD delante de cada definicion. Los finales de linea y el sangrado aparecen automaticamente


Nota: Use OL y LI para listas ordenadas UL y LI para listas no ordenadas y DL, DT y DD para listas de definicion

    
Etiquetas y atributos HTML

        

  

    Etiqueta
    Atributo
    Funcion
  

  

    <OL>....</OL>
     
    Lista ordenada (numerada)
  

  

     
    TYPE="......"
    El tipo de numeracion que se necesita para etiquetar la lista
  

  

     
    START="......"
    El valor con el cual comienza la lista
  

  

    <UL>.....</UL>
     
    Una lista no ordenada (con viñeta)
  

  

     
    TYPE="....."
    El simbolo deviñeta especifico que se utiliza para etiquetar este elemento.
  

  

    <LI>
     
    Un elemto de lista debe utilizarse con <OL> o <UL>
  

  

     
    TYPE="......"
    El tipo o numero de viñeta que se necesita para etiquetar este elemento.
  

  

     
    VALUE="......"
    El valor numerico que debe tener este elemento
  

  

    <DL>....</DL>
     
    Una lista de definicion
  

  

    <DT>
     
    Un termino de definicion, como parte de una lista de definicion
  

  

    <DD>
     
    La definicion correspondiente a un termino de definicion, como parte de una lista de definicion
