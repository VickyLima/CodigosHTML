Tamaño y color de las fuentes

Para establecer el tamaño, y color de cualquier texto de una Pagina Web, utilice la etiqueta FRONT
Eemplo: FONT SIZE=5 COLOR="PURPLE"...... /FONT
El atributo SIZE puede tomar cualquier valor entre 1 (pequeñisimo) y 7 (bastante grande), siendo 3 el tamaño normal predeterminado
El atributo COLOR puede tomar cualquier de los dos siguientes nombres estandar: Black (negro), White (blanco), Red (rojo), green (verde), blue (azul), yellow (amarillo), aquian (cian) etc.

Las etiquetas BIG y SMALL que ya se explicaron, en "Formando alineacion de texto", produce efectos que tambien pueden lograr con FRONT SIZE que le ofrece un control mas preciso sobre el tamaño pero Big y SMALL pueden trabajan con algunos navegadores Web antiguos que no soportan la etiqueta FONT

Seleccion de un tipo de letra

Una forma muy poderosa de control de fuentes: el atributo FONT FACE. El codigo para establecer el tipo de letra que se utilizo en los encabezados
Ejemplo: FONT FACE="Copperplate Gothic Bold" SIZE=5 COLOR="Red"
Tambien puede especificar una fuente de segunda opcion para quien no tenga instalada en su computadora su fuente de primera opcion. Por ejemplo yo utilize la etiqueta siguiente para elegir el texto del cuerpo
 FONT FACE ="Luicida Handwriting, Comic Sans Ms" SIZE=4 COLOR="maroon"
Las etiquetas FONT le ofrecen control sobre el color, y tipo de letra de cualquier texto

Etiquetas y atributos cubiertos de la Unidad II

Etiqueta Atributo Funcion
<FONT>......</FONT>   Controla la apariencia del texto que encierra
  SIZE="........" Agrega el tamaño fuente de 1 a 7
  COLOR="........." Cambia el color del texto
  FACE="......." Nombre de la fuente que se debe usar si hallara el nombre del usuario
<BASEFONT>   Define el tamaño de la fuente para la pagina actual
  SIZE="....." El tamaño predeterminado de la fuente del 1 al 7
